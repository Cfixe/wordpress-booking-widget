== Description ==
Cfixé Booking Widget is a WordPress extension that allows easy integration of the Cfixé appointment booking widget into WordPress sites.

= Features =
- Simple integration of the Cfixé appointment booking widget
- Customizable settings for widget slug and colors
- Supports shortcode [cfixe_booking_widget] for easy placement on pages

== Installation ==
1. Upload the `cfixe-booking-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the "Cfixé Booking Widget" plugin through the 'Plugins' menu in WordPress.
3. Create and configure your Cfixé account on [cfixe.com](https://cfixe.com).
4. Configure the widget settings under "Widget Cfixé" in the WordPress dashboard.
5. Use the shortcode `[cfixe_booking_widget]` in your pages to display the widget.

== Frequently Asked Questions ==
= How do I customize the widget settings? =
Navigate to the "Widget Cfixé" section in the WordPress dashboard and adjust the settings for the widget slug and colors.
